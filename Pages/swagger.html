Swagger

GET http://localhost:4001/team16/vendedor Obtener todos los vendedores.
POST http://localhost:4001/team16/vendedor Crear un nuevo vendedor.
Parameters: {NúmeroVendedor, Nombre, Apellido, Correo, Teléfono, Sucursal}
PUT http://localhost:4001/team16/vendedor/{idvendedor} Se modifica los datos del vendedor
Parameters: {NúmeroVendedor, Nombre, Apellido, Correo, Teléfono, Sucursal} Datos modificados
DELETE http://localhost:4001/team16/vendedor/{idvendedor} Se elimina un vendedor recibiendo su id como parametro

GET http://localhost:4001/team16/productos Obtener todos los productos
POST http://localhost:4001/team16/productos Crear un producto nuevo
Parameters: {nombre, categoria, fotos, descripcion, cantidad, cliente, pago}
PUT http://localhost:4001/team16/productos{idProducto} Se modifica los datos del producto
Parameters: {nombre_cliente, rfc, cfdi, metodo_pag, credito, correo} Datos editados
DELETE http://localhost:4001/team16/productos{idProductos} Se elimina un producto recibiendo su id como parametro

GET http://localhost:4001/team16/venta Obtener todas las ventas registradas
POST http://localhost:4001/team16/venta/crearVenta Crear una nueva venta
Parameters: { folio, fecha, monto_total, total_productos, cliente, metodo_pago, monto_pagado, cambio }
PUT http://localhost:4001/team16/venta/{idVenta} Se modifica los datos del producto
Parameters: { folio, fecha, monto_total, total_productos, cliente, metodo_pago, monto_pagado, cambio } Datos editados
DELETE http://localhost:4001/team16/venta/{idVenta} Se elimina una venta recibiendo su id como parametro
GET http://localhost:4001/team16/venta/sbf/folio Obtener una venta por folio

GET http://localhost:4001/team16/clientes Obtener todos los registros
POST http://localhost:4001/team16/clientes Crear un cliente nuevo
Parameters: {nombre_cliente, rfc, cfdi, metodo_pag, credito, correo}
PUT http://localhost:4001/team16/clientes/{idCliente} Se modifica los datos del cliente
Parameters: {nombre_cliente, rfc, cfdi, metodo_pag, credito, correo} Datos editados
DELETE http://localhost:4001/team16/clientes/{idCliente} Se elimina un cliente recibiendo su id como parametro
